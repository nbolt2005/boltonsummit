Noah Bolton Personal Site
=========================

Open index.html in your browser or VS Code Live Server.

SETUP
-----
1. Drop this folder directly into VS Code.
2. Place your images inside: assets/images/
3. Keep all image filenames exactly as referenced in the HTML.
